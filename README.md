# 🍼 Bebé IA

Un asistente de inteligencia artificial que nace desde cero y crece contigo. Sin APIs externas, sin dependencias de terceros. Solo PyTorch y amor.

## 🌱 Filosofía

Este proyecto nace de la idea de que una IA debe crecer orgánicamente, como un ser vivo:
- **Recién nacido**: Solo sabe emitir sonidos aleatorios
- **Bebé**: Aprende tus palabras y gestos básicos  
- **Niño**: Forma oraciones, entiende contexto simple
- **Adolescente**: Razona, cuestiona, tiene personalidad propia
- **Adulto**: Especialista en lo que tú le enseñaste
