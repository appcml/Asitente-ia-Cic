# Checkpoints del Bebé IA

Aquí se guardan los "cerebros" del bebé en diferentes etapas.

## Formato
- `model.pt` - Modelo actual (siempre el más reciente)
- `bebe_checkpoint_N.pt` - Checkpoint después de N interacciones
- `tokenizer.json` - Vocabulario aprendido

## Git LFS
Los archivos `.pt` se manejan con Git LFS por su tamaño.
